## 1. Architecture decision

Next.js 14 App Router on Vercel, TypeScript, Tailwind. Static JSON data at `web/public/data/forecasts/*.json` generated by `scripts/precompute_forecasts.py`. Zero runtime Python. Charts via `recharts`. Map via hand-rolled SVG over IEEE 123 BusCoords.
